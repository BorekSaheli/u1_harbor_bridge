# u1_harbor_bridge

## Installation

1. Install uv:
```bash
# Windows
winget install uv

# macOS
brew install uv

# or via pip
pip install uv
```

2. Setup environment:


```bash
uv sync

# Activate virtual environment
# Unix/macOS
source .venv/bin/activate

# Windows PowerShell
.venv\Scripts\Activate.ps1
```
